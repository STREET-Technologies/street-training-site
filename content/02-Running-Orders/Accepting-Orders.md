# Accepting orders

**In short:** Accept each new order as soon as it comes in. There is a short window to respond, so treat a new order as something to action straight away.

## Why timing matters

STREET sells on speed. The customer is expecting their order within hours, so the clock starts the moment the order lands.

- Accept the order in the Partner app as soon as you see it.
- The sooner you accept, the sooner the rider can be lined up.
- Make sure you will notice a new order arriving during your operating hours.

## If you cannot fulfil it

- If something is genuinely out of stock, do not leave the order sitting. Handle it promptly so the customer is not left waiting.

> Watch out: there is only a short time to accept before an order can be offered to another store. Action a new order right away rather than coming back to it later.
